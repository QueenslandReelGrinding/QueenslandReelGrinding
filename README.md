# Queensland Reel Grinding

Single-page brand/landing site for Queensland Reel Grinding — a specialist reel-mower
grinding service for golf courses, sports fields, councils, schools and turf managers
across Queensland and Northern NSW.

## Hosting

- **Source:** this repo (GitHub).
- **Hosting:** GitHub Pages, served from the `main` branch root.
- **Domain:** https://queenslandreelgrinding.com.au (set via the `CNAME` file; DNS managed at GoDaddy).

## Editing the site

The live site is a static page — no build step required:

- `index.html` — the page
- `assets/css/style.css` — styles
- `assets/images/` — images

Edit, commit, and push to `main`. GitHub Pages redeploys automatically within a minute or two.

> `queensland-reel-grinding-SEND.html` is a self-contained (inlined) version for emailing — it is **not** the live site.
> The `.tooling/` folder (build/screenshot scripts) is git-ignored and not deployed.

See [PRODUCT.md](PRODUCT.md) for the brand system and content brief.
